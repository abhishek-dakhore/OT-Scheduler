About Our Project

The Hospital OT Scheduling System is a smart, dynamic web application aimed at optimizing operation theater management. Built with HTML, CSS, JavaScript, Firebase, and jQuery, it bridges the gap between static manual schedules and real-time, flexible solutions.

Smart Scheduling

Automatically handle emergencies, cancellations, and updates efficiently while tracking surgical events.

Admin & User Modules

Separate dashboards for administrators and doctors for better workflow and security.

Firebase Integration

All data is managed in real-time using Firebase Authentication and Realtime Database.

Meet the Team

This system was developed as part of our college final year project by a passionate group of students committed to improving healthcare technology.
